# Carouzell Builder — GitHub Split Version

This is the split-up GitHub-ready version of the Carouzell Builder prototype.

## How to open

Open `index.html` in your browser.

## Folder structure

```text
carouzell-builder-github/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── app.js
└── assets/
    ├── forms/
    ├── coats/
    ├── markings/
    ├── eyes/
    ├── soulmarbles/
    ├── poles/
    ├── ribbons/
    ├── tack/
    ├── charms/
    ├── fins/
    └── extras/
```

## Next recommended step

Start adding finished transparent PNGs into the matching `assets/` folders. Once the art assets are ready, the placeholder CSS shapes can be replaced with real image layers.

## Notes

- This version keeps the current prototype logic in `js/app.js`.
- The visual styling is in `css/style.css`.
- The empty asset folders are included so the project is ready for GitHub organization.
